Critical Errors — Block Calculation

E-01: NovaCare RehabWorks Unit Swap — kWh Entered as Therms
NovaCare Health Partners | NC-026 (Marietta, GA) & NC-027 (Augusta, GA)
Critical

What's Wrong

Two RehabWorks acquisition clinics report electricity = 0 kWh and natural gas = 82,000 / 74,000 therms. The median NovaCare clinic uses ~5,800 therms of gas. These values are 12–14x the median.

The notes field confirms: "RehabWorks acquisition — data migrated from prior owner format."

During data migration from the prior owner's format, electricity consumption in kWh was entered into the natural_gas_therms column. The values (82,000 and 74,000) are consistent with mid-size clinic electricity consumption.

Impact If Not Fixed

Using 82,000 + 74,000 = 156,000 therms as natural gas would add ~829 mt CO2e to NovaCare's Scope 1 emissions (at 0.00531 mt/therm). NovaCare's actual total Scope 1 from gas is likely ~900 mt. This error alone would nearly double the reported Scope 1 — a 100% overstatement.

Proposed Fix

Correction
FacilityFieldCurrentCorrected
NC-026electricity_kwh082,000
NC-026natural_gas_therms82,000~3,800 (estimate)
NC-027electricity_kwh074,000
NC-027natural_gas_therms74,000~3,400 (estimate)

Rationale

Move values from therms to kWh column. For gas, apply the NovaCare Georgia clinic median (~3,600 therms, based on NC-024 Atlanta, NC-025 Savannah) as a proxy estimate. Flag as Grade C data.

Validation Required

Request actual utility bills from RehabWorks-origin facilities to confirm corrected values. Contact: Michael Park, CFO.

E-02: CSV Formatting Error — Extra Comma in 6 Rows
Harvest Ridge Foods (HRF-002, 003, 004) & Ridgeline Property Services (RPS-001, 002, 003)
Critical

What's Wrong

Six rows in the submitted CSV have 19 columns instead of 18 (extra trailing comma). This causes all columns after the extra comma to shift right by one position when parsed.

Affected fields:

  • data_source parses as blank (actual value shifted to notes column)
  • Ridgeline fleet_gasoline_gallons shifts into the data_source column — fleet data is lost
  • Harvest Ridge data_source ("utility_bill") lands in the notes column

Impact If Not Fixed

Any automated CSV parser will misattribute data source grades for these 6 facilities (appearing as "no source" instead of "utility_bill"). Ridgeline fleet gasoline values (4,200 + 2,800 + 1,200 = 8,200 gal) are silently dropped from Scope 1 calculation.

Proposed Fix

Correction
Remove the extra comma from each of the 6 affected rows. Validate column alignment after fix by checking that data_source contains a valid enum value (utility_bill, accounting_export, landlord_data, fleet_log, estimate, N/A) for every row.

Rationale

This is a data entry / export artifact. The actual values are present but misaligned. Fix is mechanical — no judgment required.

Validation Required

Can fix internally. Run column-count validation on full CSV after correction. Confirm all 139 rows have exactly 18 columns.

E-05: Patriot Fleet Gasoline Anomaly — 76,700 gal / 8 Vehicles
Patriot Staffing Solutions | All 26 facilities (fleet_gasoline column)
High

What's Wrong

The CSV reports 76,700 gallons of fleet gasoline distributed across 26 facilities (2,000–6,000 gal each). The questionnaire attributes this to 8 branch manager vehicles.

76,700 gal ÷ 8 vehicles = ~9,600 gal/vehicle/year. A typical sedan driven 15,000 miles/year at 25 mpg consumes ~600 gallons. 9,600 gal/vehicle is 16x normal.

Impact If Not Fixed

If the 76,700 gal is correct, Patriot's fleet Scope 1 emissions would be ~680 mt CO2e — a significant portion of their total footprint. If it's overstated by 10x, we'd overreport by ~600 mt.

Proposed Fix

Three hypotheses to investigate
  1. Fleet is larger than 8 vehicles. The questionnaire says "8 branch manager vehicles" but the CSV allocates fuel to 26 facilities. Patriot may have pool/shared vehicles at each branch not classified as a "fleet."
  2. Values are monthly, not annual. 76,700 ÷ 12 = ~6,400 gal/yr, or ~800 gal/vehicle — this would be plausible.
  3. Shuttle bus fuel is included. Questionnaire mentions shuttle buses at 3 locations but says fuel data is "unavailable." It may have been included in the fleet gasoline line.

Validation Required

Request fuel card statements from Scott Henderson (VP of Risk). Need: total gallons purchased in 2025, vehicle count, vehicle type breakdown. This error cannot be resolved from the data in the environment.

Medium

What's Wrong

Request landlord utility data or account access for all 6 locations. These are leased spaces — landlords have the data. Coordinate as part of upcoming lease renewals where possible.
