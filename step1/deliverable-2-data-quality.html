Scoring Methodology

This assessment evaluates data readiness for Scope 1+2 emissions calculation across Northwood's portfolio using the firm's own Climate Data Collection Protocol (Q1 2026). The protocol defines a three-tier data source hierarchy, portco-level quality thresholds, and facility-level grading rules that we apply consistently to every submission.

The assessment cross-references two primary sources — the submitted carbon CSV (139 facility-level energy rows) and each portco's climate questionnaire (narrative responses) — to validate internal consistency and flag discrepancies.

Northwood Data Source Hierarchy

Per the protocol, every data point in a portco submission carries a quality grade based on its source. Grade C data triggers a note in the portco's emissions baseline and board materials.

Source GradeDefinitionEstimation Required?
Grade A — Utility Bills Actual consumption data from utility invoices. Metered, time-stamped, verifiable. Preferred source. No estimation required.
Grade B — Accounting Estimates Derived from financial records (e.g., natural gas expense ÷ average rate per therm). Includes fleet telematics, fuel card statements, and accounting exports. Acceptable with methodology documentation.
Grade C — Landlord / Proxy Data Landlord-reported building averages, EPA benchmarks applied to square footage, or industry intensity factors applied to revenue. Last resort. Must be flagged as proxy data in all downstream analysis.
N/A — No Data Facility exists in the operational boundary but no consumption data of any kind is available. Cannot calculate. Must resolve before baseline.

Portco-Level Data Quality Rating

The protocol aggregates facility-level source grades into a portco-level quality rating. This rating appears in the emissions baseline, board materials, and GP dashboard. LP disclosures reference the portfolio-level data quality distribution.

Portco RatingThresholdImplication
● Grade A Portco>80% of emissions calculated from Grade A (utility bill) dataBaseline is defensible. Suitable for LP disclosure and exit materials.
● Grade B Portco>80% from Grade A + Grade B combinedBaseline is usable with documented methodology notes. Upgrade path to A is clear.
● Grade C Portco>20% of emissions rely on Grade C (proxy/landlord) dataBaseline carries quality caveat. Flagged in board materials. Data improvement plan required.

Readiness Score

In addition to the protocol rating, we assign a 1–5 readiness score that factors in source grade distribution, cross-source consistency, detected errors, boundary clarity, and temporal coverage.

ScoreLabelMeaning
5Calculation-ReadyGrade A portco with no cross-source discrepancies. Data can go directly into emissions calculation.
4Minor RemediationGrade A or B portco with minor issues (formatting, small discrepancies). Fixable internally.
3Moderate RemediationGrade B portco or data with notable gaps. Baseline possible with caveats. Follow-up data request likely needed.
2Significant RemediationGrade C portco or data with material errors. Baseline unreliable without additional data from portco.
1Not CalculableMajor errors or gaps that block analysis entirely. Data would produce misleading results if used.
